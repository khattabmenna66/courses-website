Course Overview

Data Structuresis a method of organizing, storing, and managing data so it can be accessed and modified efficiently within a program. Programming languages, such as Python, Java, or C++, are tools used to implement data structures and algorithms.

What You Will Learn

  • A data structure is a named location that can be used to store and organize data. And, an algorithm is a collection of steps to solve a particular problem
  • This course will get you started with Data Structures and Algorithms (DSA) and help you master them from the basics
  • Understanding what is a data structure and why do we need it, understanding algorithms and related concepts
  • Getting hands-on with some most common data structures such as Arrays, Linked Lists, Stack, Queue, and understanding how they work
